What is css?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper or in other media.
  3. CSS is not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML Documents in 3 ways:

  • Inline - by using he style attribute inside HTML elements.
  • Internal - by using a style element in head section
  • External - by using a link element to link to an external css file
  • The most common way to add CSS, is to keep styles in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies styles to multiple elements. ID on other hand applies style to one unique element. ID is also special in that you can use a special URL to link directly to an element and its used by Javascript

In CSS, selectors are used to target a specific element or range of elements on a webpage. Once an element is targeted, a style or set of styles can be applied to the element.

There are wide range of selectors available. two of most commonly used are class and ID. Both are used to target elements to which a style should be applied.
